# Sentimental-analysis-using-nlp
* This project is about analyzing the reviews of the customer in E-commerce website
  E-Commerce websites hosts millions of customer reviews, which contain valuable insights about customer sentiment across a wide range of products. However, manually analyzing these reviews to understand customer 
  feedback at scale is challenging and inefficient.
